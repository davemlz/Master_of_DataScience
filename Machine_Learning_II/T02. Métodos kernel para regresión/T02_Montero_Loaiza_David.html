from sklearn.datasets import fetch_openml

def load_mauna_loa_atmospheric_co2():
    ml_data = fetch_openml(data_id=41187, cache=False)
    months = []
    ppmv_sums = []
    counts = []

    y = ml_data.data[:, 0]
    m = ml_data.data[:, 1]
    month_float = y + (m - 1) / 12
    ppmvs = ml_data.target

    for month, ppmv in zip(month_float, ppmvs):
        if not months or month != months[-1]:
            months.append(month)
            ppmv_sums.append(ppmv)
            counts.append(1)
        else:
            # aggregate monthly sum to produce average
            ppmv_sums[-1] += ppmv
            counts[-1] += 1

    months = np.asarray(months).reshape(-1, 1)
    avg_ppmvs = np.asarray(ppmv_sums) / counts
    return months, avg_ppmvs

X, y = load_mauna_loa_atmospheric_co2()

from sklearn.gaussian_process import GaussianProcessRegressor
from sklearn.gaussian_process.kernels import RBF, ConstantKernel as C, WhiteKernel
from sklearn.gaussian_process.kernels import RationalQuadratic
from sklearn.gaussian_process.kernels import ExpSineSquared

from time import time

X = np.array(data.months).reshape(-1,1)
y = data.avg_ppmvs

k1 = C(50, (1e1, 1e+3)) * RBF(50,(1e0,1e5))  # long term smooth rising trend
k2 = C(5, (1e1, 1e+2)) * RBF(100,(1e0,1e5)) \
    * ExpSineSquared(length_scale=1.0, length_scale_bounds=(1e-1,1e1),
                     periodicity=1.0, periodicity_bounds="fixed")  # seasonal component
k3 = C(0.5, (1e-2, 1e1)) * RationalQuadratic(length_scale=1.0, length_scale_bounds=(1e-2,1e3),
                                alpha=1.0, alpha_bounds=(1e-1,1e3)) # medium term irregularities
k4 = C(0.1, (1e-3, 1e+1)) * RBF(0.1,(1e-2,1e2)) \
    + WhiteKernel(noise_level=0.1**2,noise_level_bounds=(1e-4, 1e0)) # noise terms

kernel = k1 + k2 + k3 + k4

gp = GaussianProcessRegressor(kernel=kernel, alpha=0,normalize_y=True, n_restarts_optimizer=1)

t0 = time()

gp.fit(X,y)

print("Elapsed time: %0.3fs"%(time() - t0))

print("\nLearned kernel: %s" % gp.kernel_)
print("Log-marginal-likelihood: %.3f" % gp.log_marginal_likelihood(gp.kernel_.theta))

X_ = np.linspace(X.min(), X.max() + 20, 1000)[:,np.newaxis]
y_pred, y_std = gp.predict(X_, return_std=True)

Elapsed time: 48.350s

Learned kernel: 31.6**2 * RBF(length_scale=28) + 3.16**2 * RBF(length_scale=164) * ExpSineSquared(length_scale=1.49, periodicity=1) + 0.733**2 * RationalQuadratic(alpha=0.285, length_scale=1.25) + 0.187**2 * RBF(length_scale=0.117) + WhiteKernel(noise_level=0.0406)
Log-marginal-likelihood: -187.348

Hiperparámetros hasta diciembre de 2001 (LML = -115.438):

    31.6**2 * RBF(length_scale=45.6)
    + 3.16**2 * RBF(length_scale=97.1) * ExpSineSquared(length_scale=1.58, periodicity=1)
    + 0.536**2 * RationalQuadratic(alpha=3.05, length_scale=0.967)
    + 0.189**2 * RBF(length_scale=0.122) + WhiteKernel(noise_level=0.0367)

Hiperparámetros con todos los datos (LML = -187.348):

    31.6**2 * RBF(length_scale=28)
    + 3.16**2 * RBF(length_scale=164) * ExpSineSquared(length_scale=1.49, periodicity=1)
    + 0.733**2 * RationalQuadratic(alpha=0.285, length_scale=1.25)
    + 0.187**2 * RBF(length_scale=0.117) + WhiteKernel(noise_level=0.0406)

  • La señal objetivo se mantuvo en 31.6 ppm, aunque la tendencia creciente a largo plazo se vio explicada por una escala de 28 años.

  • La componente periódica también mantuvo su amplitud de 3.16 ppm pero con un deterioro de 164 años y una escala de 1.49.

  • El ruido correlacionado aumentó una amplitud de 0.733 ppm con una escala de 1.25 años.

  • El ruido blanco aporta un total de 0.187 ppm.

Si bien los hiperparámetros no son siempre los mismos, se asemejan en gran medida. El ruido correlacionado aumentó y esto pudo provocar la disminución del LML del modelo. No obstante, la cantidad de ruido aportada no es muy alta y se puede obtener una buena confianza en el modelo generado.
